After installation, you will find a base structure like the one described below:

```
app
├── components
│  ├── <Component>
│  │  ├── collections
│  │  ├── models
│  │  ├── resolvers
│  │  ├── http
│  │  ├── jobs
│  │  └── services
├── helpers
└── services
    └── http
    └── jobs
    └── graphql
    └── echoes
```
